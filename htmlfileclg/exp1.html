Course Curriculum: Web Development Basics

Objectives

  1. To design a simple course curriculum webpage using HTML.
  2. To practice the use of basic text formatting tags (bold, italic, underline, and special characters like <, >, &).
  3. To learn the implementation of ordered, unordered, description, nested, and mixed lists.
  4. To understand webpage content structuring using headings and paragraphs.

Course Description

This course introduces students to the fundamentals of web development.
The topics covered include HTML basics, page structuring, and the use of different lists for organizing content.
Students will also practice writing clean and structured code for simple webpages.

Course Topics

  • Introduction to HTML
  • Text Formatting
    • Bold Text
    • Italic Text
    • Underline Text
    • Special Characters (<, >, &, ©, ®)
  • Lists in HTML
    1. Ordered List
    2. Unordered List
    3. Description List
    4. Nested and Mixed Lists
  • Structuring Webpage Content

Course Modules

Module 1: Basics of HTML
- Covers introduction, tags, and attributes.
Module 2: Formatting
- Using bold, underline, and special characters.
Module 3: Lists
- Implementing ordered, unordered, description, nested, and mixed lists.
Module 4: Page Structuring
- Using headings, paragraphs, and semantic layout.
